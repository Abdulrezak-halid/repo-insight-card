## What It Does

- Fetches public GitHub profile data.
- Fetches owned repository data.
- Generates `dist/profile-card.svg`.
- Generates `dist/insights.json`.
- Updates this README automatically.

## GitHub API Usage

This project integrates with the GitHub REST API:

| Endpoint                      | Purpose                                     |
| ----------------------------- | ------------------------------------------- |
| `GET /users/{username}`       | Reads public profile data.                  |
| `GET /users/{username}/repos` | Reads public repository data for analytics. |

The API client is in `src/github.js`.

## Usage

Run locally:

```bash
GITHUB_USERNAME=Abdulrezak-halid npm run generate
```

Run tests:

```bash
npm test
```

GitHub Actions runs the generator automatically on pushes, on a daily schedule, and manually from the Actions tab.

## Configuration

| Variable          | Description                                                                     |
| ----------------- | ------------------------------------------------------------------------------- |
| `GITHUB_USERNAME` | GitHub username to analyze. Defaults to the repository owner in GitHub Actions. |
| `GITHUB_TOKEN`    | Optional token for authenticated GitHub API requests.                           |
| `OUTPUT_DIR`      | Output directory. Defaults to `dist`.                                           |
| `README_PATH`     | README path. Defaults to `README.md`.                                           |
