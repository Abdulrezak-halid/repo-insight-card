# Contributing

Thanks for improving Repo Insight Card.

## Development

1. Use Node.js 20 or newer.
2. Copy `.env.example` to `.env` and set `GITHUB_USERNAME`.
3. Run `npm test`.
4. Run `npm run generate`.

## Pull requests

- Keep changes focused.
- Add or update tests when behavior changes.
- Update documentation when configuration, output, or GitHub API usage changes.
- Do not commit personal access tokens or private account data.
